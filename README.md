# Open PDF Tools

Open PDF Tools is an open-source web application that provides essential PDF utilities such as:

- Merge PDFs
- Split PDFs
- Compress PDF
- PDF to Images
- Images to PDF

## Tech Stack
- Backend: Node.js, Express
- Frontend: React (planned)
- PDF Processing: pdf-lib

## Disclaimer
This project is an independent open-source tool and is **not affiliated with Sejda PDF or any other commercial PDF service**.
