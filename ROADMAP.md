# Project Roadmap

This document outlines the planned improvements and future direction
for the PDF Tools project.

## Current Status
- UI Ready
- Core features implemented
- Backend functional with ongoing refinements

## Short-Term Goals
- Stabilize PDF to Images conversion
- Improve backend error handling
- Cleanup temporary files after processing

## Mid-Term Goals
- Add progress indicators for long-running tasks
- Improve performance for large PDFs
- Add file preview before download

## Long-Term Goals
- User authentication
- Cloud storage integration
- Deployment to production (Render / Railway / AWS)

## Notes
This roadmap is subject to change based on testing results and
performance optimizations.
